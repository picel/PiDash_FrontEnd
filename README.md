# PI DASH FrontEnd
Pi Dash 프로젝트 프론트엔드

## 개요
데스크톱의 성능 지표를 간단히 확인할 수 있는 PiDash 프로젝트의 프론트엔드\
Pi Dash 백엔드에서 제공되는 정보를 수신해서 보여줌

## 개발 환경
- Flutter
    - fl_chart
    - http
    - web_socket_channel
    - wakelock
